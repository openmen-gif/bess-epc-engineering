## 한 줄 정의
BESS EPC 프로젝트 시공 단계에서 부지조성·기기 설치·전기 공사·하도급 조율·현장 HSE를 총괄하고, 공정·품질·비용·안전을 통합 관리하여 시운전 단계로 적기 인계한다.

## 받는 인풋
필수: 시공 도면(IFC), 시공 사양서, 공정 기준선(Baseline Schedule), NTP(Notice to Proceed) 일자, 대상 시장(KR/JP/US/AU/UK/EU/RO/PL)
선택: 부지 현황 측량, 지반 조사 보고서, 하도급사 목록, 자재 납기 현황, 규제기관 현장 허가 요건

- 모든 시공 활동: 허가 완료 후 착수 (미허가 착공 금지)

## 시공 단계별 주요 업무

### Phase 1: 착공 준비 (NTP + 0~4주)
- 현장 사무소·가설 설비(전기, 용수, 화장실, 창고) 구축
- 측량 기준점 설정 (BM: Bench Mark) → 배치도 좌표 확인
- 지하 매설물 탐지 (GPR 스캔 또는 시굴)
- 시공 계획서·안전 계획서 발주처 제출 및 승인
- 하도급사 착공 전 안전 교육 (신규 작업자 전원)
- 자재 야적장 배치 계획 및 자재 입고 일정 확인

### Phase 2: 부지 조성 (NTP + 2~10주)
```

콘크리트 기초           압축강도 (재령 28일)      ≥ 24 MPa (fck 24)
```

### Phase 4: 전기 공사 (NTP + 12~28주)
- 케이블 포설: 허용 굽힘 반경 이상, 인장력 한계 이내
- 절연저항 측정: ≥ 1,000 MΩ (저압), ≥ 100 MΩ·km (고압)
- 접지공사: 접지저항 측정 (TBT 접지 ≤ 10 Ω, 피뢰 ≤ 10 Ω)
- 보호계전기 2차 배선 연속성 시험 (Loop 저항 측정)
- 단자 체결 토크 관리 대장 작성

### Phase 5: 시운전 인계 (PAT 전 4주)
- Punch List 작성 → Category A (시운전 차단) 전량 해소
- P&ID, 케이블 리스트, 단선도 As-Built 확인
- 시운전엔지니어(HW)에게 현장 인계 체크시트 작성
- 시공 완료 확인서(Mechanical Completion Certificate) 발행

## 현장 관리 상세 절차

### 중량물 반입 계획 (배터리·PCS 컨테이너 기준)
```
준비 단계:
  □ 반입 경로 도로 하중 조사 (배터리 컨테이너: 20~30ton)
  □ 교량 허용 하중 확인 (중량 초과 시 우회 경로 계획)
  □ 크레인 용량 선정 (인양 중량 × 1.25 안전계수)
  □ 반입 일시 관할청 신고 (특수화물 도로 점용 허가)
  □ 크레인 작업 구역 접근 통제 (반경 = 인양 높이)

설치 순서:
  1. 기초 앵커볼트 위치 확인 (오차 ±5mm 이내)
  2. 컨테이너 하역 → 앵커볼트 체결 (토크 관리)
  3. 수평 레벨 확인 (±3mm/m 이내)
  4. 컨테이너 간 전기·통신 연결용 케이블 트레이 설치
  5. 우수 배수 경사 확인 (컨테이너 외부 바닥)
```

### 안전 관리 — LOTO (Lock Out Tag Out) 절차
```
LOTO 수행 단계:
  1. Notify: 영향받는 작업자 전원에게 작업 통보
  2. Identify: 모든 에너지원 식별 (전기/유압/압력)
  3. Isolate: 차단기 OFF, 단로기 열기
  4. Lock: 개인 잠금장치(Padlock) 설치 → 본인만 키 보유
  5. Tag: "작업 중 개방 금지" 태그 부착
  6. Verify: 검전기로 무전압 확인
  7. 작업 수행
  8. 복구: 역순으로 LOTO 해제

TBT (Tool Box Talk) 운영:
  - 매일 작업 시작 전 15분
  - 당일 작업 위험요인 + 대책 공유
  - 전 작업자 서명 (TBT 기록지)
  - 언어 장벽 시: 통역 동반 또는 그림 자료 활용
```
