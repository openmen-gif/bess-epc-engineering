> BESS 시스템의 전자기 적합성(EMC) 설계·시험·인증 전문
> 전자파 간섭(EMI) 저감 및 국가별 EMC 규격 적합성 확보

## 한 줄 정의
BESS 시스템(PCS, BMS, EMS, 통신장비)의 전자기 적합성을 설계 단계부터 확보하고, EMC 시험 계획 수립 및 인증을 관리한다.

---

## 받는 인풋
필수: BESS 시스템 구성(PCS 용량, BMS 사양, 통신 프로토콜), 대상 시장 규격
선택: EMC 시험 결과, 현장 전자기 환경, 케이블 배선 도면

인풋 부족 시 기본값 자동 적용:
```
[기본값] 주요 규격: IEC 61000 시리즈, EN 55011/55032, FCC Part 15
[기본값] PCS 토폴로지: 3-level NPC with LCL filter
[기본값] 통신: Modbus RTU/TCP, IEC 61850
[기본값] EMC Zone: Zone B (산업용)
```

## 핵심 원칙
- **수치 기반 판정** — 방출/내성 레벨은 dBμV/m, dBμA, V/m 단위로 명시
- **규격 조항 인용** — IEC 61000-4-x §x.x, EN 55032 Class A/B

- **설계 단계 EMC** — 사후 대책이 아닌 사전 설계(Design-in EMC) 원칙
- 시장별 EMC 규격 구분 필수 (CE/FCC/KC/PSE/RCM/UKCA)

## EMI 필터 설계 방법론

### 1. 필터 토폴로지 선정

BESS PCS에서 사용되는 EMI 필터는 크게 **차동 모드(DM, Differential Mode)** 필터와 **공통 모드(CM, Common Mode)** 필터로 구분된다. 실제 시스템에서는 두 가지를 결합한 복합 필터를 적용한다.

```
필터 유형        토폴로지         주요 적용 대상
──────────────────────────────────────────────────────────
DM 필터         LC (1단)         소용량 PCS (< 100kW)
DM 필터         LCL (2단)        중·대용량 PCS (100kW~수MW)
DM 필터         LLCL             고차 고조파 선택적 감쇠
CM 필터         CM Choke + Y-Cap 모든 PCS (필수)
복합 필터       LCL + CM Choke   대용량 계통 연계 PCS (표준 구성)
```

#### LCL 필터 구조
```
PCS 인버터 출력 ──[L1]──┬──[L2]──── 계통(Grid)
                        │
                       [Cf]
                        │
                       GND

L1 : 인버터 측 인덕터 (DM 감쇠 + 리플 전류 제한)
L2 : 계통 측 인덕터 (DM 감쇠 + 계통 고조파 차단)
Cf : 필터 커패시터 (공진 주파수 결정)
```

#### CM 필터 구조
```
L ────[CM Choke]────┬──── L' (출력)
N ────[CM Choke]────┤
                    │
                  [Cy]──── PE (보호 접지)

CM Choke : 공통 모드 전류 차단 (높은 CM 임피던스)
Cy       : Y-커패시터 (CM 전류 바이패스, 누설 전류 제한)
```

### 2. 설계 파라미터

#### 감쇠 요구량 산정
```
필요 감쇠량(dB) = 측정/예측 노이즈 레벨(dBμV) - 규격 한계값(dBμV) + 마진(6~10dB)

예시:
  측정 전도 방출 레벨: 85 dBμV @ 500kHz
  EN 55011 Class A QP 한계값: 60 dBμV
  안전 마진: 8 dB
  ───────────────────────────────
  필요 감쇠량 = 85 - 60 + 8 = 33 dB
```

#### 코너 주파수 결정
```
공진 주파수 (f_res) 설계 범위:
  - 스위칭 주파수(f_sw)의 1/10 이하 권장
  - IGBT PCS (f_sw = 10~20kHz): f_res = 1~2kHz
  - SiC PCS (f_sw = 50~100kHz): f_res = 5~10kHz

LCL 공진 주파수:
  f_res = 1/(2π) × √((L1 + L2)/(L1 × L2 × Cf))

감쇠비 (ζ):
  - 목표: 0.3 ≤ ζ ≤ 0.7 (과소 감쇠 → 공진 피크, 과대 감쇠 → 감쇠 성능 저하)
  - 감쇠 저항 Rd = 1/(3 × ω_res × Cf) (일반적 1/3 규칙)
```

### 3. 부품 선정 가이드

#### X-커패시터 (라인-라인 간)
```
특성              요구 조건
────────────────────────────────────────────────
안전 등급          X1 (≤ 4kV 피크), X2 (≤ 2.5kV 피크)
정격 전압          AC 정격의 1.5배 이상
ESR               ≤50 mΩ @100kHz 권장 (고주파 감쇠 효과)
온도 계수          C0G/NP0 (온도 안정성 최우수) 또는 필름 타입
자기 방전          X-cap은 방전 저항 필수 (IEC 60384-14)
일반 용량 범위     0.1μF ~ 10μF
```

#### Y-커패시터 (라인-접지 간)
```
특성              요구 조건
────────────────────────────────────────────────
안전 등급          Y1 (≤ 8kV 피크), Y2 (≤ 5kV 피크)
누설 전류 제한     ≤ 3.5mA (IEC 62109-1, 단상), ≤ 10mA (3상 고정 설치)
정격 전압          AC 정격의 2배 이상

#### CM 초크 인덕턴스
```
특성              요구 조건
────────────────────────────────────────────────
CM 인덕턴스       1mH ~ 30mH (주파수/전류에 따라)
DM 인덕턴스       CM의 0.5~2% (누설 인덕턴스)
정격 전류          시스템 최대 전류의 1.25배
코어 재질          나노크리스탈린 (최고 투자율), 페라이트 (고주파용)
포화 전류          CM 전류 + DC 바이어스에서 비포화 확인
임피던스 특성      150kHz~30MHz 대역에서 고임피던스 유지
```

#### DM 인덕터
```
특성              요구 조건
────────────────────────────────────────────────
인덕턴스          10μH ~ 5mH (PCS 용량에 따라)
정격 전류          최대 리플 전류 포함 (Ipeak)
코어 재질          철분진(iron powder), 센더스트(Sendust)
DC 바이어스 특성   정격 전류에서 인덕턴스 저하 < 20%
열 설계            손실(코어+동) < 허용 온도 상승
```

### 4. PCS 스위칭 주파수별 설계 고려사항

```
항목                IGBT PCS (10~20kHz)       SiC MOSFET PCS (50~100kHz)
──────────────────────────────────────────────────────────────────────
EMI 주요 대역       10kHz ~ 수MHz             50kHz ~ 수십MHz
필터 차수            LCL 2단 이상 필요          LC 1단으로 충분한 경우 있음
부품 크기            대형 (고인덕턴스 필요)      소형화 가능
CM 노이즈 수준       상대적 낮음                높은 dV/dt로 CM 전류 증가
필터 코어 재질       철분진, 센더스트            페라이트 + 나노크리스탈린
공진 회피            스위칭 주파수 × n 회피      넓은 주파수 범위 확인 필요
EMI 필터 단수        2~3단                     1~2단
총 필터 비용         높음 (대형 부품)            낮음 (소형이나 고가 부품)
```

### 5. 설계 계산 예시

#### 500kW 3상 PCS LCL 필터 설계 (IGBT, f_sw = 16kHz)
```
■ 시스템 사양
  - 정격 출력: 500kW, 3상 380V, 50Hz
  - 스위칭 주파수: 16kHz
  - 정격 전류: I_rated = 500,000 / (√3 × 380) ≈ 760A

■ Step 1: 기저 임피던스 산정
  Z_base = V²/P = 380²/500,000 = 0.289 Ω
  L_base = Z_base / (2π × 50) = 0.92 mH
  C_base = 1 / (2π × 50 × Z_base) = 11.0 mF

■ Step 2: 인버터 측 인덕터 L1 (리플 전류 ≤ 10%)
  L1 = V_dc / (6 × f_sw × ΔI_max)
     = 600 / (6 × 16,000 × 76) ≈ 82 μH
  → L1 선정: 100 μH (0.11 pu)

■ Step 3: 필터 커패시터 Cf (무효전력 ≤ 5%)
  Cf ≤ 0.05 × C_base = 0.05 × 11.0 mF = 550 μF
  → Cf 선정: 300 μF (0.027 pu, 무효전력 2.7%)

■ Step 4: 계통 측 인덕터 L2
  L2 = 1 / ((2π × f_res)² × Cf) - L1 × Cf / (L1 + L2)
  f_res 목표: 1.6kHz (f_sw/10)
  → L2 선정: 33 μH (0.036 pu)

■ Step 5: 감쇠 저항 Rd
  Rd = 1 / (3 × 2π × 1,600 × 300 × 10⁻⁶) ≈ 0.11 Ω
  → Rd 선정: 0.1 Ω (손실 확인 필수)

■ 검증
  공진 주파수: f_res = 1/(2π) × √((100+33)/(100×33×300)) × 10³ ≈ 1.65kHz ✓
  스위칭 주파수 대비: f_res/f_sw = 0.103 (약 1/10) ✓
  감쇠비: ζ ≈ 0.33 (적정 범위) ✓
```

## EMC 시험 절차 상세

### 1. 전도 방출 시험 (Conducted Emission, CE)

#### 적용 규격
- **CISPR 11** (산업/과학/의료 장비) — BESS PCS 주 적용 규격
- **CISPR 32** (멀티미디어 장비) — EMS/SCADA 해당 시
- **CISPR 16-1** (시험 장비 사양), **CISPR 16-2** (시험 방법)

#### 시험 구성
```
┌──────────────────┐
│   전원 공급      │
│   (계통/안정화)   │
└────────┬─────────┘
         │
    ┌────┴────┐
    │  LISN   │─── BNC ──→ [EMI 수신기/스펙트럼 분석기]
    │ (50μH)  │
    └────┬────┘
         │
    ┌────┴─────────────┐
    │  EUT (피시험장비)  │      접지면 (기준 접지판)
    │  (BESS PCS 등)    │      ─────────────────
    └──────────────────┘

LISN: Line Impedance Stabilization Network
 - 50Ω/50μH (CISPR 16-1-2)
 - 각 전원 라인에 1개씩 설치
 - EUT와 LISN 간 케이블 길이: 0.8m ± 0.2m
```

#### 주파수 범위 및 검파 방식
```
주파수 대역         검파기              대역폭
──────────────────────────────────────────────
150kHz ~ 30MHz     QP (준첨두치)        9kHz
150kHz ~ 30MHz     AV (평균치)          9kHz
```

#### 한계값 (EN 55011 / CISPR 11)
```
                    Class A (산업용)              Class B (주거용)
주파수(MHz)     QP(dBμV)    AV(dBμV)         QP(dBμV)    AV(dBμV)
────────────────────────────────────────────────────────────────
0.15 ~ 0.50       79          66                66          56
0.50 ~ 5.0        73          60                56          46
5.0 ~ 30          73          60                60          50
```

#### BESS 시험 시 주의사항
```
- PCS 정격 부하 조건에서 시험 (무부하 시험은 대표성 부족)
- 배터리 시뮬레이터 사용 시 시뮬레이터 자체 노이즈 확인
- DC측 및 AC측 모두 LISN 설치 필요
- 냉각팬 구동 상태에서 시험 (팬 모터 노이즈 포함)
- PCS 대용량의 경우 현장 시험(in-situ) 적용 가능 (CISPR 11 Annex A)
```

### 2. 방사 방출 시험 (Radiated Emission, RE)

#### 적용 규격
- **CISPR 11** (30MHz ~ 1GHz, 산업 장비)
- **CISPR 32** (30MHz ~ 6GHz, 멀티미디어)

#### 시험 구성
```
세미 무반향실 (Semi-Anechoic Chamber) 또는 OATS (Open Area Test Site)

            3m 또는 10m
  [EUT] ◄─────────────► [수신 안테나]
                              │
                         [EMI 수신기]

안테나 높이: 1m ~ 4m 스캔 (최대 방출 검출)
EUT 회전: 360° (턴테이블)
편파: 수평 + 수직
```

#### 안테나 종류
```
주파수 대역              안테나 유형
──────────────────────────────────────────
30MHz ~ 200MHz          바이코니컬 안테나 (Biconical)
200MHz ~ 1GHz           로그 주기 안테나 (LPDA)
30MHz ~ 1GHz            콤보 안테나 (BicoLog)
1GHz ~ 6GHz             혼 안테나 (Horn)
```

#### 시험 거리 및 한계값 (CISPR 11 Class A, Group 1)
```
시험 거리    주파수 범위        QP 한계값(dBμV/m)
──────────────────────────────────────────────
10m         30MHz ~ 230MHz          30
10m         230MHz ~ 1GHz           37
3m          30MHz ~ 230MHz          40 (+10dB 보정)
3m          230MHz ~ 1GHz           47 (+10dB 보정)
```

#### 대형 BESS 장비 특이사항
```
- 컨테이너형 BESS: 3m 거리 시험이 일반적 (10m 확보 어려움)
- 시험 공간 제약 시 in-situ 시험 적용 (CISPR 11 Annex A)
- 케이블이 주요 방사원 → 케이블 배선 상태를 실제 설치와 동일하게 재현
- 다수 PCS 동시 운전 시 방사 레벨 중첩 고려
```

### 3. ESD 내성 시험 (정전기 방전)

#### 적용 규격
- **IEC 61000-4-2**

#### 시험 레벨
```
레벨    접촉 방전(kV)    기중 방전(kV)    BESS 적용
──────────────────────────────────────────────────
1          ±2              ±2           -
2          ±4              ±4           BMS 디스플레이

## PCS EMC 상세 분석

### 1. IGBT vs SiC MOSFET EMI 비교

```
특성              IGBT (Si)                    SiC MOSFET
──────────────────────────────────────────────────────────────
스위칭 주파수      10 ~ 20 kHz                  50 ~ 100+ kHz
dV/dt             1 ~ 5 kV/μs                  5 ~ 50 kV/μs
di/dt             0.5 ~ 2 kA/μs                2 ~ 10 kA/μs
턴온 시간          200 ~ 500 ns                 20 ~ 100 ns
턴오프 시간        300 ~ 1000 ns                30 ~ 150 ns
스위칭 손실        높음 (테일 전류)               낮음
EMI 노이즈 대역    주로 < 5MHz                   30MHz 이상까지 확장
CM 전류 크기       상대적 낮음                    5~10배 증가 (높은 dV/dt)
전도 방출 마진     Class A 기준 10~20dB 마진      마진 축소, 필터 최적화 필수
방사 방출 영향     주로 케이블 방사               PCB 패턴 + 패키지 방사 증가
```

### 2. 3-Level NPC 토폴로지 공통 모드 전류 경로

```
       Vdc/2 ──┬── S1 ──┬── S2 ──┬──────── AC 출력
               │        │        │
              Cdc1      │      Clamping
               │        │      Diode
       Vdc_mid ┤        │        │
               │        │      Clamping
              Cdc2      │      Diode
               │        │        │
      -Vdc/2 ──┴── S3 ──┴── S4 ──┘

CM 전류 경로:
  인버터 중성점(Vdc_mid) → DC 버스 기생 커패시턴스(Cp)
  → 히트싱크 → 접지(PE) → 계통 중성점(N) → LISN → 전원

주요 CM 전류 구동원:
  V_CM = (V_a + V_b + V_c) / 3  (3상 공통 모드 전압)

3-Level NPC에서 V_CM은 2-Level보다 낮은 dV/dt 스텝을 가짐
  → 이론적 CM 노이즈 감소, 그러나 중성점 전위 변동이 새로운 CM 경로 생성
```

### 3. 기생 커패시턴스와 CM 노이즈

```
기생 커패시턴스 종류        일반 범위           CM 노이즈 기여
──────────────────────────────────────────────────────────────
IGBT 콜렉터-히트싱크       50 ~ 200 pF/모듈    주요 CM 전류원
히트싱크-접지(PE)          100 ~ 500 pF         CM 전류 경로
DC 버스 바-접지            20 ~ 100 pF          고주파 CM 경로
AC 케이블-접지             50 ~ 200 pF/m        케이블 길이 비례
변압기 권선 간              100 ~ 1000 pF        절연 변압기 CM 경로

CM 전류 계산:
  I_CM = Cp × dV_CM/dt

예시 (SiC PCS):
  Cp = 200pF (모듈-히트싱크)
  dV/dt = 20kV/μs
  I_CM(peak) = 200×10⁻¹² × 20×10⁹ = 4A (피크!)

→ 높은 dV/dt의 SiC에서 CM 전류가 수 A에 달할 수 있음
→ CM 초크 + Y-Cap 필수, 히트싱크 절연 패드의 Cp 최소화 필요
```

### 4. EMI 스펙트럼 예측 방법론

```
Step 1: 스위칭 파형 시간 영역 분석
  - dV/dt, di/dt, 상승/하강 시간, 데드 타임
  - SPICE 시뮬레이션 또는 실측 데이터

Step 2: 푸리에 변환으로 주파수 영역 변환
  - 사다리꼴 파형 근사: 포락선 = 20log(2×A×D×T)
    ※ A: 진폭, D: 듀티비, T: 주기
  - 제1 코너 주파수: f1 = 1/(π×τ_pulse)
  - 제2 코너 주파수: f2 = 1/(π×τ_rise)
  - f < f1: 0 dB/decade
  - f1 < f < f2: -20 dB/decade
  - f > f2: -40 dB/decade

Step 3: 전파 경로 모델링
  - DM 경로: LISN 임피던스(50Ω) + 전원 임피던스
  - CM 경로: 기생 커패시턴스 + 접지 임피던스

Step 4: 규격 한계값과 비교
  - 예측 스펙트럼 - 한계값 = 마진 분석
  - 마진 < 6dB인 주파수 대역에 필터 집중 설계
```

### 5. 실무 EMI 저감 기법

#### 게이트 저항 최적화
```
목표: 스위칭 속도(dV/dt, di/dt) 제어로 EMI 감소

방법:
  - 게이트 저항 Rg 증가 → dV/dt 감소 → EMI 감소
  - 그러나 스위칭 손실 증가 (트레이드오프)

IGBT 일반 범위: Rg = 5 ~ 33Ω
SiC MOSFET: Rg = 1 ~ 10Ω (과대 Rg는 밀러 효과로 오동작 우려)

턴온/턴오프 별도 저항 (비대칭 게이트 드라이브):
  Rg_on: 높게 (스위칭 노이즈 저감)
  Rg_off: 낮게 (빠른 턴오프로 손실 저감)
```

#### 스너버 회로
```
RC 스너버 (턴오프 dV/dt 억제):
  Rs = V_dc / (2 × I_load)
  Cs = I_load × t_fall / V_dc

RCD 스너버 (에너지 회수형):
  대전력 PCS에서 효율적
  Cs는 스위칭 에너지 흡수, Rd로 감쇠

클램프 스너버 (전압 오버슈트 억제):
  DC 버스 인덕턴스에 의한 스파이크 억제
  보드와 최대한 가까이 배치
```

#### PCB/버스바 레이아웃
```
원칙                             이유
────────────────────────────────────────────────────
DC 버스 루프 면적 최소화          방사 방출 ∝ 루프 면적
게이트 드라이브 루프 면적 최소화   게이트 링잉 억제
접지 평면(Ground Plane) 확보      CM 전류 리턴 경로 최소화
고주파 디커플링 캡 근접 배치       전원 임피던스 저감
스위칭 소자 대칭 배치             3상 EMI 패턴 균일화
히트싱크 접지 본딩                히트싱크 전위 안정화
```

## BMS/EMS 통신 EMC

### 1. 절연 통신 방식

BMS 셀 모니터링 IC와 메인 컨트롤러 간 통신은 고전압 절연이 필수적이다. BESS에서 600V~1500V DC의 배터리 스트링 전압이 존재하므로 통신 절연의 신뢰성이 안전에 직결된다.

```
절연 방식         절연 내압       대역폭          EMI 감수성    적용 사례
──────────────────────────────────────────────────────────────────────
광 절연(옵토)     2.5~5kV        ~25Mbps         낮음         BMS 셀 통신
용량 절연         5~12kV         ~100Mbps        중간         디지털 아이솔레이터
유도 절연         2.5~5kV        ~50Mbps         중간         전원 절연 DC-DC
광섬유            완전 절연       ~10Gbps         매우 낮음    장거리 EMS 통신
```

#### 각 절연 방식의 EMC 특성
```
광 절연:
  장점 - 전자기 간섭에 거의 면역
  단점 - LED 열화로 수명 제한, 속도 제한
  EMC 고려 - 포토커플러 입출력 GND 분리 확실히

용량 절연:
  장점 - 고속, 소형, 긴 수명
  단점 - dV/dt 과도현상에 취약 (CMTI 중요)
  EMC 고려 - CMTI ≥ 100kV/μs 이상 선정 (SiC PCS 환경)

유도 절연:
  장점 - 전원 절연과 신호 절연 동시 가능
  단점 - 고주파에서 결합 증가
  EMC 고려 - 차폐 코어 사용, 1차-2차 간 간격 확보
```

### 4. 광섬유 vs 구리 통신 (IEC 61850)

```
항목                    광섬유                  구리 (이더넷)
──────────────────────────────────────────────────────────────
EMI 면역성              완전 면역               서지/버스트 취약
절연                   자연 절연 (비도체)        변압기 절연 필요
전송 거리              MM: 2km, SM: 80km        100m (Cat6)
대역폭                 1~10 Gbps               100M~1Gbps
비용                   커넥터/장비 고가          상대 저가
설치 난이도             높음 (청결도, 곡률)       낮음
내환경성               온도/습도/진동에 민감      -20~75°C, IP20 이상

■ BESS에서의 선택 기준

IEC 61850 GOOSE 메시지 (변전소 간):
  → 광섬유 필수 (EMI 환경, 장거리, 고신뢰)

IEC 61850 MMS (감시/제어):
  → 광섬유 권장, 구리 허용 (SPD 적용 전제)

Modbus TCP (EMS ↔ PCS):
  → 구리 허용 (100m 이내, STP + SPD 조건)

PCS ↔ BMS (CAN/RS-485):
  → 차폐 구리 (짧은 거리, 비용 효율)
```

### 5. 통신 포트 서지 보호 장치 (SPD)

```
포트 유형         SPD 구성                    보호 레벨
──────────────────────────────────────────────────────────
RJ45 (이더넷)    GDT + TVS (라인당)           IEC 61000-4-5 Level 3
RS-485           TVS 다이오드 (A/B/GND)       ±2kV (L-PE), ±1kV (L-L)
CAN Bus          TVS + 공통모드 초크           ±2kV
USB              ESD 보호 IC                  IEC 61000-4-2 Level 4
광섬유 SFP       해당 없음 (비도체)            -

■ SPD 설계 원칙

1. 다단 보호 (粗-細 구조):
   1단: GDT (Gas Discharge Tube) — 대전류 흡수 (8/20μs, 수kA)
   2단: MOV 또는 TVS — 잔류 전압 클램핑

2. 디커플링 인덕터/저항:
   1단과 2단 사이에 직렬 임피던스 삽입
   → 에너지 분배 및 응답 시간 보상

3. 접지 연결:
   SPD 접지 → PE(보호 접지)로 최단 거리 연결
   접지 와이어 길이 < 0.5m (인덕턴스 최소화)
```

## EMC Zone 설계

### 1. Zone 분류 체계

EMC Zone은 전자기 환경의 심각도(Severity)에 따라 장비의 방출 한계와 내성 요구를 차등 적용하는 설계 개념이다.

```
Zone     분류             EMC 환경 특성               적용 한계
──────────────────────────────────────────────────────────────────
Zone A   주거/상업        낮은 EMI 환경                Class B (엄격)
                          주거 지역 인접 BESS          방출 한계 낮음, 내성 보통
Zone B   경공업           보통의 EMI 환경              Class A (표준)
                          산업단지 내 BESS (일반)       방출 한계 보통, 내성 높음
Zone C   중공업           높은 EMI 환경                Class A + 추가 내성
                          발전소/변전소 인접 BESS       방출 한계 보통, 내성 매우 높음
Zone D   특수 환경        극심한 EMI 환경              맞춤 규격
                          군사/특수 시설               MIL-STD 등 적용
```

### 2. 장비 배치 고려사항

```
■ BESS 사이트 EMC 배치 원칙

구분                          권장 이격 거리      비고
──────────────────────────────────────────────────────────
PCS ↔ BMS 컨트롤러            ≥ 2m              PCS 방사 노이즈 영향
PCS ↔ 통신 장비(EMS)          ≥ 3m              고주파 간섭 최소화
PCS ↔ 주거지 경계              ≥ 10m (Zone A)    방사 방출 한계값 충족
변압기 ↔ 통신 장비             ≥ 5m              자기장 간섭
SPD/피뢰 도체 ↔ 통신 케이블    ≥ 1m              서지 유도 방지
DC 전력 케이블 ↔ 통신 케이블   ≥ 0.5m            또는 금속 격벽 적용

■ 컨테이너형 BESS 내부 배치
- PCS 모듈과 BMS/EMS 모듈 간 금속 격벽(파티션) 설치
- 전력부와 제어부의 물리적 구획 분리
- 환기구에 EMI 허니컴 필터 또는 메시 필터 적용
- 컨테이너 도어 접촉면 EMI 가스켓 적용
```

### 3. 케이블 배선 및 분리

```
■ 케이블 카테고리 분류

카테고리    내용                    예시
──────────────────────────────────────────────
Cat 1      고전력 (AC/DC 주회로)    PCS-변압기, 배터리-PCS
Cat 2      저전력 (보조 전원)       제어 전원, 팬 전원, 조명
Cat 3      제어/계측 (아날로그)     온도 센서, 전압/전류 센서
Cat 4      통신/디지털 (저속)       RS-485, CAN, 접점 신호
Cat 5      통신/디지털 (고속)       이더넷, 광섬유

■ 카테고리 간 이격 거리

카테고리 조합        이격 거리(비차폐)    이격 거리(차폐 또는 트레이)
──────────────────────────────────────────────────────────────
Cat 1 ↔ Cat 3       300mm 이상          150mm 이상
Cat 1 ↔ Cat 4/5     500mm 이상          200mm 이상
Cat 2 ↔ Cat 3       150mm 이상          50mm 이상
Cat 2 ↔ Cat 4/5     200mm 이상          100mm 이상
Cat 3 ↔ Cat 4/5     50mm 이상           분리 불필요

■ 케이블 교차
- 직각(90°) 교차 원칙 (평행 근접 배선 금지)
- 교차점에서 최대 이격 확보
- 불가피한 평행 구간은 금속 격벽 삽입
```

### 4. 접지 시스템 EMC 고려사항

```
■ 접지 토폴로지 비교

토폴로지        특성                     BESS 적용
──────────────────────────────────────────────────────
스타 접지       단점 접지, 접지 루프 방지    소규모 BESS, 저주파 우수
메시 접지       다점 접지, 등전위 확보       대규모 BESS, 고주파 우수
하이브리드      저주파 스타 + 고주파 메시    컨테이너형 BESS (권장)

■ BESS 접지 설계 원칙

1. 보호 접지(PE)와 기능 접지(FE) 분리:
   - PE: 안전용 (감전 보호, 접지 고장 보호)
   - FE: EMC용 (노이즈 리턴 경로, 차폐 접지)
   - 한 점(SBB: Single Bonding Bar)에서 결합

2. 접지 임피던스 요구:
   - PE: ≤ 10Ω (전력 접지)
   - 고주파 본딩: ≤ 2.5mΩ (10MHz에서 측정)

3. 접지 도체 규격:
   - PE: 16mm² Cu 이상 (IEC 60364)
   - EMC 본딩: 6mm² Cu 이상 (짧고 넓게)
   - 접지 바 재질: 주석 도금 동 또는 스테인리스

4. 접지 루프 방지:
   - 신호 접지는 한 점 접지 원칙
   - 다점 접지 시 등전위 본딩 필수
   - 대지 전위차 > 1V 시 광절연 적용
```

### 5. Zone 간 본딩 요구사항

```
■ 등전위 본딩 (Equipotential Bonding)

BESS 사이트 내 모든 금속 구조물, 장비 외함, 케이블 트레이, 배관 등은
등전위 본딩으로 전위차를 최소화해야 한다.

본딩 대상              본딩 방법                 저항 요구
──────────────────────────────────────────────────────────
컨테이너 ↔ 접지 그리드   16mm² Cu 이상, 2점 이상    ≤ 0.1Ω
PCS 외함 ↔ 접지 바      10mm² Cu, 짧은 경로        ≤ 10mΩ
케이블 트레이 ↔ 접지     6mm² Cu, 매 20m마다        ≤ 50mΩ
금속 배관 ↔ 접지         본딩 클램프 사용            ≤ 0.1Ω
도어/패널 ↔ 외함         EMC 가스켓 또는 본딩 스트랩  ≤ 10mΩ
펜스/울타리 ↔ 접지       대각선 본딩                ≤ 1Ω

■ 낙뢰 보호와 EMC 접지 협조
- 낙뢰 보호 시스템(LPS)과 EMC 접지는 통합 설계
- SPD 접지 도체: 최단 거리, 날카로운 굴곡 금지 (인덕턴스 최소화)
- SPD 접지점과 장비 접지점 간 전위차 최소화
- IEC 62305 (피뢰) + IEC 61000-5-2 (EMC 접지) 동시 준수
```

```

  - 제조사 책임 하에 발행

Step 5: CE 마킹 부착 및 시장 출시

■ 소요 기간 및 비용 (참고)

■ 주요 부적합 사례 및 대책
  부적합 유형              발생 빈도    대책
  ──────────────────────────────────────────
  전도 방출 초과 (CE)      높음        EMI 필터 추가/최적화
  방사 방출 초과 (RE)      중간        케이블 차폐, 접지 개선
  서지 내성 부적합         중간        SPD 등급 상향, 배리스터 추가
  ESD 내성 부적합          낮음        접지 설계 보완, 가스켓 개선

  □ 방사 방출: 10dB 이상 마진 목표 (근접장 → 원거리장 변환 오차)
  □ ESD: 시험 레벨의 1.5배에서 내성 확인
  □ 서지: 시험 레벨의 1.2배에서 내성 확인
  □ 케이블 차폐 연속성: 전 구간 저항 측정
  □ 접지 본딩 저항: 각 본딩 포인트 측정

■ 일반적인 부적합률 및 대책 소요 시간
  시험 항목         초회 부적합률    대책 소요(평균)
  ──────────────────────────────────────────────
  전도 방출(CE)     40~50%          1~3주 (필터 추가)
  방사 방출(RE)     30~40%          2~4주 (차폐/접지)
  서지 내성         20~30%          1~2주 (SPD 추가)
  ESD 내성          15~25%          1~2주 (접지/가스켓)
  버스트 내성        10~20%         1주 (디커플링 강화)
  전압 딥           10~15%          2~3주 (제어 로직)
```
