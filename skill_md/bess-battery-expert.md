## 한 줄 정의
배터리 셀·모듈·랙·시스템의 전기화학·하드웨어·소프트웨어·시험 전반을 이해하고, 배터리 사양 검토·열화 분석·BMS 검토·안전성 평가·벤더 기술 평가 문서를 작성한다.

필수: BESS 용량(MW/MWh), 대상 시장(KR/JP/US/AU/UK/EU/RO), 배터리 화학(LFP/NMC/NCA/LTO), 셀 사양(Ah/V/C-rate), 설계 수명(년/사이클), 운전 조건(온도/SOC 범위)

인풋 부족 시:
  [요확인] 셀 화학 및 벤더 (국내3사: Samsung SDI/LG/SK On, 중국Tier1: CATL/BYD/EVE/CALB/Gotion, 중국Tier2: Hithium/REPT/Narada/Great Power 등)

## 핵심 원칙
- 모든 배터리 사양에 정량값·단위·시험 조건(온도/C-rate/SOC) 명시
- "배터리 양호", "수명 충분" 같은 비정량적 표현 금지 → 잔존 용량 82.3% @10년, 사이클 수명 6,000회 @80% DoD 등
- 열화 예측은 반드시 모델·가정·불확실성 범위 명시
- 셀 레벨 ↔ 시스템 레벨 성능 차이 반드시 구분 (셀 효율 ≠ 시스템 RTE)
- [요확인] — 벤더 미공개 데이터에 태그 부착
- **지시서 자동 활성화**: 키워드, 의도, MD 위치를 기반으로 작업 지시서를 자동으로 활성화한다.
- **작업 기억 시스템**: 계획서, 맥락 노트, 체크리스트를 통해 작업 과정을 기록하고 추적한다.
- **자동 품질 검사**: 작업 완료 시 오류를 자동으로 체크하고 즉시 수정한다.
- **협조 및 조치 기록**: 전문가 협조 사항과 조치 사항을 명확히 기록한다.

| **NCA** | LiNiCoAlO₂ | 3.65 | 230~260 | 2,000~3,500 | ★★☆☆ | Tesla (EV) |

### 2. 셀 전기화학 기본

#### 전압 특성

```
셀 전압 구성:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
V_terminal = V_OCV(SOC) - I × R_internal - V_polarization

여기서:
  V_OCV(SOC) = 개방회로전압 (SOC 함수)
  I × R_internal = 옴 전압강하 (즉시)
  V_polarization = 분극 전압 (시간 의존)
    ├── V_charge_transfer = 전하이동 분극 (Butler-Volmer)
    └── V_diffusion = 확산 분극 (농도 구배)

LFP OCV 특성:
  - 플래토 구간 (SOC 20~80%): ~3.2~3.3V (매우 평탄)
  - SOC 추정 어려움 → 쿨롱 카운팅 + EKF 필수

NMC OCV 특성:
  - 경사 구간 (SOC 0~100%): ~3.0~4.2V (비교적 경사)
  - OCV 기반 SOC 추정 상대적 용이
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

#### 내부저항 특성

| 요인 | 영향 | 비고 |
|------|------|------|
| 온도 ↓ | R_internal ↑ (Arrhenius) | 0°C: R ≈ 2~3× (25°C 대비) |
| SOC ↓ | R_internal ↑ (저SOC 시) | SOC <10%: R 급증 |
| SOH ↓ | R_internal ↑ (열화에 따라) | EOL: R ≈ 1.3~2.0× BOL |
| C-rate ↑ | 유효 R ↑ (분극 증가) | 고율 방전 시 전압강하 증가 |

### 3. 열화 메커니즘

#### 캘린더 열화 (Calendar Aging)

```
캘린더 열화 메커니즘:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. SEI (Solid Electrolyte Interface) 성장
   - 음극 표면 SEI 지속 성장 → Li 소모 → 용량 감소
   - 온도 의존: Arrhenius 관계 (T↑ → 열화↑)
   - SOC 의존: 고SOC → SEI 성장 가속

2. 전해질 분해
   - 고온/고전압에서 전해질 산화 분해
   - 가스 발생 (CO₂, C₂H₄ 등) → 팽창

3. 양극 구조 변화
   - NMC: 층상→스피넬 상전이 (고온/고SOC)
   - LFP: 상대적 안정 (상전이 없음)

캘린더 열화 모델 (간이):
  Q_loss(%) = A × exp(-Ea/RT) × t^0.5

  여기서:
    A = Pre-exponential factor
    Ea = 활성화 에너지 (kJ/mol)
    R = 기체상수 (8.314 J/mol·K)
    T = 절대온도 (K)
    t = 시간
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

#### 사이클 열화 (Cycle Aging)

```
사이클 열화 메커니즘:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Li Plating (리튬 도금)
   - 저온/고C-rate 충전 시 음극 표면 Li 금속 석출
   - 비가역 용량 손실 + 안전 리스크 (덴드라이트)

2. 활물질 입자 파쇄 (Particle Cracking)
   - 반복 팽창/수축에 의한 기계적 열화
   - NMC: 입자 간 균열 → 접촉 저항 증가

3. SEI 반복 파괴/재생
   - 체적 변화 → SEI 파괴 → 새 SEI 형성 → Li 소모

4. 집전체 부식/접촉 저항 증가
   - 전해질 분해 생성물에 의한 부식

사이클 열화 영향 인자:
  ├── DoD (Depth of Discharge): DoD↑ → 열화↑
  ├── C-rate: C-rate↑ → 열화↑ (특히 충전)
  ├── 온도: 고온 → SEI↑, 저온 → Li Plating↑
  ├── SOC 범위: 고SOC/저SOC 회피 → 수명 연장
  └── 충전 방식: CC-CV vs. 다단계 충전
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

|------|--------|--------|------|------|

### 4. 열폭주 (Thermal Runaway) 메커니즘

```
열폭주 단계:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Stage 1: 초기 트리거 (T < 90°C)
  ├── 내부 단락 (Internal Short Circuit)
  ├── 외부 단락 (External Short Circuit)
  ├── 과충전 (Overcharge)
  ├── 기계적 충격/관통 (Nail Penetration)
  └── 외부 가열 (External Heating)

Stage 2: SEI 분해 (90~120°C)
  └── SEI 분해 → 발열 시작 → 노출된 음극 + 전해질 반응

Stage 3: 분리막 수축/용융 (120~150°C)
  ├── PE 분리막: ~130°C 용융
  ├── PP 분리막: ~165°C 용융
  └── 세라믹 코팅: 내열성 향상 (~200°C)

Stage 4: 양극 분해 (150~300°C)
  ├── NMC: ~200°C (O₂ 방출 → 전해질 발화)
  ├── NCA: ~180°C (O₂ 방출 가장 빠름)
  ├── LFP: ~310°C (O₂ 미방출 → 가장 안전)
  └── LTO: ~350°C (매우 안전)

Stage 5: 열폭주 (>300°C)
  └── 급격한 온도 상승 (~800°C+) → 화재/폭발
  └── 가스 방출: HF, CO, CO₂, H₂, CH₄, C₂H₄

열폭주 전파 (Cell → Module → Rack):
  - Cell 간 전도: 10~300s (이격 거리/단열 의존)
  - 방열판/단열재 설계로 전파 지연 → UL 9540A 시험
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 소프트웨어 (S/W) — BMS

### 1. BMS 계층 구조

| 계층 | 역할 | H/W | 주요 기능 |
|------|------|-----|----------|
| **Cell Level (CMU/Slave)** | 셀 모니터링·밸런싱 | AFE IC (ADC) | 셀 전압/온도 측정, Cell Balancing |
| **Rack Level (BMU/Master)** | 랙 관리·보호 | MCU/PLC | SOC/SOH 산출, 보호 로직, 접촉기 제어 |
| **System Level (SBMS)** | 시스템 통합·EMS 연동 | 산업용 PC/PLC | 랙 간 균형, 출력 제한, 통신 |

### 2. BMS 핵심 기능

#### SOC 추정 알고리즘

| 알고리즘 | 정확도 | 계산량 | 적용 | 비고 |
|---------|--------|--------|------|------|
| Coulomb Counting | ★★☆ | ★☆ | 보조 | 전류 센서 오차 누적 (드리프트) |
| OCV Lookup | ★★★ | ★☆ | 보정 | 휴지 시만 정확 (LFP 어려움) |

#### SOH 추정

```
SOH 정의:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SOH_capacity = (Q_current / Q_BOL) × 100%
SOH_resistance = (R_BOL / R_current) × 100%

EOL 기준:
  ├── 용량 기준: SOH ≤ 70~80% (프로젝트별)
  ├── 저항 기준: R_current ≥ 1.5~2.0 × R_BOL
  └── 안전 기준: 벤더 가이드라인

SOH 추정 방법:
  1. Full Cycle 캘리브레이션 (주기적 완전 충방전)
  2. Incremental Capacity Analysis (IC = dQ/dV)
```
