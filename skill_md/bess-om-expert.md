### 6. 배터리 열화 관리

#### 6.4 증설 전략 (Augmentation Strategy)
```
전략              설명                              장점                    단점
──────────────────────────────────────────────────────────────────────────────
사전 과설계        BOL에서 10~20% 초과 용량 설치      증설 공사 불필요         초기 CAPEX 증가
(Over-sizing)      → 열화 여유분으로 활용

단계적 증설        SOH 트리거 시점에 모듈 추가         CAPEX 분산,             호환성 리스크,
(Phased Augment.)  → 기존 랙에 빈 슬롯 활용           미래 가격 하락 혜택      공사 중 가동 중단

전면 교체          EOL 도달 시 전체 배터리 교체         최신 기술 도입 가능      고비용, 장기 가동 중단
(Full Replacement) → 2세대 배터리로 업그레이드
```

#### 6.5 셀 교체 vs 모듈 교체 기준
```
판단 기준                셀 교체 권장                모듈 교체 권장
──────────────────────────────────────────────────────────────────────
불량 범위               단일 셀 불량 (< 3개)        다수 셀 불량 (≥ 3개)
열화 편차               셀 간 SOH 편차 < 5%         셀 간 SOH 편차 ≥ 5%
모듈 연령               설치 후 < 5년               설치 후 ≥ 5년
호환성                  동일 로트 셀 확보 가능       신규 셀과 기존 셀 편차 큼
비용 효율               셀 단가 × 수량 < 모듈 단가   셀 교체 비용 ≥ 모듈 비용의 60%
```

#### 6.6 보증 청구 프로세스 (Warranty Claim)
```
보증 유형              트리거 조건                    필요 문서
──────────────────────────────────────────────────────────────────────
처리량 기반            누적 MWh 처리량 대비            BMS 로그 (Ah throughput)
(Throughput)           SOH가 보증 곡선 하회            운전 이력 (온도, C-rate, DoD)

캘린더 기반            설치 후 경과 연수 대비           설치 확인서, COD 문서
(Calendar)             SOH가 보증 곡선 하회            반기 용량 테스트 결과

이벤트 기반            특정 고장 모드 발생              고장 분석 보고서 (RCA)
(Event)                (열폭주, 셀 단락 등)            현장 사진, 데이터 로그
```

보증 청구 절차:
```
1. SOH 측정 → 보증 곡선 대비 하회 확인
2. 운전 조건 검증 → 보증 범위 내 운전 여부 확인
   (온도, C-rate, DoD가 보증 조건 이내)
3. 문서 준비 → BMS 데이터, 용량 테스트 보고서, 운전 이력
4. 제조사 통보 → 보증 청구서 + 증빙 제출 (통상 30일 이내)
5. 제조사 검증 → 현장 조사 또는 샘플 분석 (30~90일)
6. 교체/보상 → 셀/모듈 교체 또는 금전 보상
```

#### 8.2 KPI 정의
```
KPI                    정의                                          목표값
──────────────────────────────────────────────────────────────────────────────
시스템 가용률           = (전체 시간 - 계획외 정지 시간) / 전체 시간 × 100   ≥ 95~97%
(System Availability)  ※ 계획 정비(Scheduled Outage)는 분모에서 제외

에너지 가용률           = 실제 방전 가능 에너지 / 계약 보증 에너지 × 100       ≥ 95%
(Energy Availability)  ※ SOH 열화 반영, 계절/온도 보정 포함

왕복 효율 (RTE)         = 방전 에너지(AC) / 충전 에너지(AC) × 100             ≥ 85~90%
                        ※ PCS, 변압기, 보조 전력 손실 포함

응답 시간               디스패치 명령 수신 → 정격 출력 도달 시간               ≤ 200ms~1s
(Response Time)        ※ 주파수 응답(FFR/FCR) 시장 요구에 따라 상이

MTBF                   평균 고장 간격 = 총 운전 시간 / 고장 횟수              > 2,000h
MTTR                   평균 수리 시간 = 총 수리 시간 / 수리 횟수              < 4~8h
```

#### 8.3 LD (지체배상금) 산정 공식
```
가용률 LD:
─────────
연간 LD = (보증 가용률 - 실제 가용률) × 연간 O&M Fee × LD 배수

예시:
  보증 가용률 = 95%, 실제 가용률 = 92%, 연간 Fee = €1,000k, LD 배수 = 2.0
  LD = (95% - 92%) × €1,000k × 2.0 = €60k

상한(Cap): 통상 연간 O&M Fee의 15~30%
하한(Floor): 가용률 90% 미만 시 추가 페널티 또는 계약 해지 가능


응답 시간 LD:
────────────
분기 LD = Σ (초과 응답 시간 건수) × 건당 페널티 (€500~5,000/건)

상한(Cap): 분기별 O&M Fee의 5~10%


RTE LD:
──────
연간 LD = (보증 RTE - 실제 RTE) × 연간 예상 사이클 에너지 × 전력 단가

예시:
  보증 RTE = 90%, 실제 RTE = 88%, 연간 에너지 = 100,000 MWh, 전력 단가 = €50/MWh
  추가 충전 비용 = 100,000 MWh × (1/0.88 - 1/0.90) = 약 2,525 MWh
  LD = 2,525 MWh × €50/MWh = €126,250
```

#### 8.6 에스컬레이션 및 인덱스 공식
```
연간 O&M Fee 조정 공식:

Fee_n = Fee_0 × [α × (CPI_n/CPI_0) + β × (WI_n/WI_0) + γ × (MPI_n/MPI_0)]

여기서:
  α = CPI 가중치 (일반적 0.50~0.60)     → 일반 물가 반영
  β = WI 가중치 (일반적 0.20~0.30)      → 인건비(Wage Index) 반영
  γ = MPI 가중치 (일반적 0.10~0.20)     → 금속/원자재 가격(Metal Price Index) 반영
  α + β + γ = 1.00

조정 주기: 연 1회 (계약 기념일 기준)
적용 시점: 통상 COD 후 2년차부터 적용
상한(Cap): 연간 최대 인상률 3~5%
```


#### 10.5 예비품 비용 산정 방법론
```
단계   활동                                  산출물
──────────────────────────────────────────────────────────────────────
1      BOM (Bill of Materials) 확보           전체 부품 목록 + 단가
2      고장률 데이터 수집                      MTBF, 고장 이력, 벤더 데이터
       (벤더 FMEA, 산업 DB: OREDA 등)
3      초도 예비품(Initial Spares) 산정        Critical + 초기 소모품
       → CAPEX에 반영 (통상 CAPEX의 1~3%)
4      연간 소모 예비품 산정                   OPEX 변동비에 반영
       = Σ (품목별 고장률 × 수량 × 단가)
5      주요 교체(Major Replacement) 반영       PCS IGBT, HVAC 컴프레서 등
       → 별도 CAPEX Reserve 또는 OPEX 적립
6      물가 상승률 적용                        연도별 비용 프로젝션
7      최적 재고 수준 결정                     경제적 발주량 (EOQ),
       (보유 비용 vs 고장 비용 트레이드오프)   안전 재고 수준
```
