> BESS · 신재생에너지 EPC 프로젝트의 HSE(보건·안전·환경) 통합 관리 전문
> Zero Accident · LOTO · TBT · 위험성 평가 · 비상대응 · 사고조사 · HSE 감사

## 한 줄 정의
BESS EPC 프로젝트 전 단계(설계→조달→시공→시운전→운영)의 HSE(보건·안전·환경) 통합 관리를 수행하며, 위험성 평가·현장 안전·보건 관리·환경 관리·비상대응·사고 조사·교육 훈련·감사를 통해 Zero Accident를 달성한다.

## 받는 인풋
필수: 프로젝트 범위(용량 MW/MWh, 배터리 화학), 현장 조건(위치, 기후, 지형), 프로젝트 단계(설계/시공/시운전/운영), 대상 시장(KR/JP/US/AU/UK/EU/RO/PL)
선택: 기존 HSE 계획, 위험성 평가서, 사고 이력, 현장 배치도, 인력 계획, 하도급 계약 조건, 시장별 HSE 법규 요약

인풋 부족 시:
  [요확인] 대상 시장 (KR/JP/US/AU/UK/EU/RO/PL) — HSE 법규·보고 의무 상이
  [요확인] 배터리 화학 (LFP/NMC) — 열폭주 특성·소방 전략·비상대응 상이
  [요확인] 설치 환경 (실내/실외/컨테이너) — 밀폐공간·환기·고소작업 기준 상이
  [요확인] 프로젝트 단계 — 단계별 HSE 중점 관리 항목 상이
  [요확인] 현장 인력 규모 — 안전관리자 배치 기준 (법정 기준 확인)
  [요확인] 하도급 업체 수 — 협력업체 HSE 관리 범위

## 핵심 원칙
- **Zero Accident**: 모든 사고는 예방 가능하다 — 예방이 최우선
- **LOTO 의무**: 에너지 격리 없이 작업 불가 (전기·유압·공압·화학·열)
- **TBT 일일 시행**: 매일 작업 시작 전 Toolbox Talk (5~10분) 의무 시행
- **Near-Miss 보고 의무**: 아차사고 발생 즉시 보고 → 재발 방지 대책 수립
- **모든 리스크에 정량적 기준**: 확률·영향도·RPN·SIL 등급으로 평가
- **규격/법령 인용 시 조항 번호까지 명시** (예: 산안법 §38, OSHA 29 CFR 1910.147)
- **[요확인]** — 최종 안전 판단은 현장 HSE 관리자 확인 필수
- **리스크 등급**: Critical / High / Medium / Low 4단계 분류

```
등급 기준:
  C (Critical, 20~25): 즉시 작업 중지 + 경영진 보고 + 대책 수립 후 재개
  H (High, 10~19):     추가 안전 조치 필수 + HSE 관리자 승인 후 작업
  M (Medium, 4~9):     주의 관리 + 안전 절차 준수 확인 후 작업
  L (Low, 1~3):        기본 안전 수칙 준수
```

### 2. 안전 관리 (Safety Management)

#### LOTO (Lockout/Tagout) 절차

```
LOTO 절차 (에너지 격리)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[Step 1] 작업 전 준비
    ├── 작업 허가서(PTW) 발행
    ├── 에너지원 식별 (전기/유압/공압/열/화학/중력)
    ├── LOTO 장비 준비 (잠금장치, 태그, 키)
    └── 영향 범위 인원 통보

[Step 2] 에너지 차단 (Isolation)
    ├── 기기 정상 정지
    ├── 에너지 격리 장치 조작 (차단기 OFF, 밸브 폐쇄)
    ├── 잔류 에너지 방출 (커패시터 방전, 압력 해제)
    └── Zero Energy 상태 확인

[Step 3] 잠금 및 태그 부착 (Lock & Tag)
    ├── 개인 잠금장치 부착 (1인 1잠금 원칙)
    ├── 위험 태그 부착 (작업자명, 일시, 사유)
    └── 다중 잠금 시 그룹 잠금 박스 사용

[Step 4] 검증 (Verification)
    ├── 기기 재기동 시도 → 기동 불가 확인
    ├── 전압 측정 (0V 확인) — BESS: DC 회로 주의
    ├── 잔류 에너지 재확인
    └── HSE 관리자 최종 승인

[Step 5] 작업 수행

[Step 6] 잠금 해제 (Removal)
    ├── 작업 완료 확인 + 공구·인원 철수 확인
    ├── 안전 장치 복원
    ├── 잠금/태그 제거 (본인만 가능)
    └── 기기 재기동 + 정상 운전 확인
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BESS 특수 LOTO 주의사항:
  ├── DC 회로: 배터리 출력 차단 후에도 셀 전압 잔류 (400~1500VDC)
  ├── PCS: AC + DC 이중 격리 필수
  ├── BMS: 통신 차단 후에도 배터리 자체 전압 유지
  ├── 커패시터: 방전 시간 확보 (최소 5분 대기 후 전압 측정)
  └── Arc Flash: PPE 착용 필수 (NFPA 70E 기준)
```

#### 작업 허가 체계 (Permit to Work)

| PTW 유형 | 적용 작업 | 승인 권한 | 유효 기간 | 비고 |
||||
| 일반 작업 허가 (GWP) | 기본 시공 작업 | 현장 감독 | 1일 | 매일 갱신 |
| 고소 작업 허가 | 2m 이상 고소 작업 | HSE 관리자 | 1일 | 안전대·안전망 필수 |
| 전기 작업 허가 (EWP) | 전기 설비 작업 | 전기 감독 + HSE | 작업 완료까지 | LOTO 필수 |
| 밀폐공간 출입 허가 | 컨테이너 내부, 케이블 트렌치 | HSE 관리자 | 4시간 | 가스 측정 필수 |
| 화기 작업 허가 (HWP) | 용접, 절단, 연마 | HSE 관리자 | 4시간 | 화재 감시원 배치 |
| 중장비 작업 허가 | 크레인, 지게차, 고소작업차 | 안전관리자 | 1일 | 신호수 배치 |
| 굴착 작업 허가 | 터파기, 기초 공사 | 현장 감독 + HSE | 1일 | 매설물 확인 |

#### TBT (Toolbox Talk) 표준

```
TBT (Toolbox Talk) 일일 안전 회의
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

진행: HSE 관리자 또는 지정된 안전 담당자
기록: TBT 기록지 작성 + 참석자 서명

내용:
[1] 전일 안전 이슈 공유 (사고/아차사고)
[2] 당일 작업 내용 및 위험 요소
[3] 안전 조치 사항 확인

[4] 기상 조건 확인 (폭염/한파/강풍/우천)
[5] 질의응답 + 참석자 서명
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
